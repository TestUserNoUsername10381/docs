**Forge** is a Lua/Luau script protection platform that transforms readable source into a heavily virtualized, encrypted runtime bundle. Every build is unique — randomized opcodes, randomized dispatch, per-build keys.

## What Forge does

<CardGroup cols={2}>
  <Card title="Virtualization" icon="cpu">
    Compiles your Lua into a custom VM with a unique instruction set per build.
  </Card>

  <Card title="String Encryption" icon="lock">
    Removes all plaintext strings; decrypted lazily at runtime.
  </Card>

  <Card title="Control Flow Flattening" icon="git-branch">
    Replaces structured branches with a state-machine dispatcher.
  </Card>

  <Card title="Anti-Tamper" icon="shield-check">
    Integrity checks, opcode validation, and runtime heartbeats.
  </Card>
</CardGroup>
