The Forge CLI lets you protect scripts from your terminal or CI pipeline.

## Install

```bash
npm install -g @forge/cli
```

## Authenticate

```bash
forge login
```

Opens your browser to sign in and stores a token in `~/.forge/credentials`.

## Protect a file

```bash
forge protect input.lua --preset balanced --target luau --out protected.lua
```

## Common flags

| Flag | Description |
| --- | --- |
| `--preset <name>` | `fast`, `balanced`, `strong`, `maximum`, `vm-standard`, `vm-hardened`, `vm-extreme` |
| `--target <runtime>` | `luau`, `lua51`, `lua52`, `lua53`, `lua54`, `luajit` |
| `--output-length <level>` | `low`, `medium`, `high`, `super-high`, `ultra-high` |
| `--out <path>` | Output file path |
| `--watermark <bool>` | Override watermark (paid plans only) |
| `--config <path>` | Load a saved preset from disk |

## Exit codes

| Code | Meaning |
| --- | --- |
| 0 | Success |
| 1 | Build failed |
| 2 | Authentication required |
| 3 | Insufficient tickets |
| 4 | Invalid input |
