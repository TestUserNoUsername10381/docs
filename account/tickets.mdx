Forge measures usage in **tickets**. Each protect or build operation costs a configurable amount of tickets based on protection strength and output length.

## Monthly allowance

| Plan | Monthly tickets |
| --- | --- |
| Free | 200 |
| Starter | 2,000 |
| Pro | 10,000 |
| Titan | 50,000 |
| Lifetime | Unlimited |

Tickets reset on the first day of each billing cycle. Lifetime is uncapped.

## Ticket costs (typical)

| Operation | Cost |
| --- | --- |
| Fast preset build | 1 |
| Balanced preset build | 2 |
| Strong preset build | 4 |
| VM Standard build | 8 |
| VM Hardened build | 16 |
| VM Extreme build | 32 |

<Note>
  Exact costs are configurable by Forge admins and may change.
</Note>

## Earning free tickets

- Daily login bonus
- Verified account bonus
- Promotions and limited-time events
